# Paradise Nursery 🌿

A React-based e-commerce shopping application for houseplants, built with Redux Toolkit for state management.

## Project Name
**Paradise Nursery** — Your green sanctuary, delivered to your door.

## Features
- 🌱 Browse 18+ unique houseplants across 3 categories
- 🛒 Redux-powered shopping cart
- ➕➖ Adjust quantities in cart
- 🗑️ Remove items from cart
- 📱 Fully responsive design

## Tech Stack
- React 18
- Redux Toolkit
- Vite
- CSS Modules

## Getting Started

```bash
npm install
npm run dev
```

## Deployment
Deployed via GitHub Pages.
